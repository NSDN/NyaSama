NyaSama
=======

这里是 [喵玉殿](http://bbs.nyasama.com/) Android 客户端的咪

## Features
与已经停止更新的 Discuz 客户端 掌上论坛 不同，
这个客户端基于原生界面重新实现了论坛客户端的基本功能。
结合后端修改过的 [DiscuzMobile](https://github.com/NSDN/DiscuzMobile) 插件，
提供的拓展功能还包括

* 帖子点评，投票，编辑，搜索等
* 用户提醒和消息通知
* 发送表情，自定板块图标，浏览图片附件的大图模式
* 论坛签到等插件的支持
* 帖子的删除，移动，置顶等（管理员）
